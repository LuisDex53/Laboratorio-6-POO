En este laboratorio disenaremos un controlador de dispositivos inteligentes, que permite accionar sobre los dispositivos de una casa inteligente.
-La casa tendra n cantidad de habitaciones, con m dispositivos inteligentes.
-Cada dispositivo podra ser apagado o encendido desde el controlador.
-Algunos dispositivos tendran comportamientos particulares, por ejemplo, en una lavadora se pueden seleccionar varios ciblos de lavado, o se puede reiniciar una computadora
-EL controlador permite tambien apagar o encender, todos los dispositivos de una habitacion en particular
